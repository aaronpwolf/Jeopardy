# Jeopardy
## TV-style Jeopardy games to use as study guides in class, as .json files

Instructions: 

(1) Make a copy of the dummy configuration file: [sample_game.json](sample_game.json)

(2) Replace dummy text with your Clues and Answers

(3) Indicate Daily Doubles with the string ```"dailyDouble": true```. (Note: .json syntax requires a comma to end the line above this string.)

Run the file by uploading via the "Choose File" button at https://jeopardy.brianyu.me/
